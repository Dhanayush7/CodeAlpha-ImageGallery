# 📸 Professional Image Gallery

A modern and responsive Image Gallery web application built using **HTML, CSS, and JavaScript**. This project showcases images from multiple categories such as Gym, Sports, WWE, Bikes, Cars, Nature, Technology, and Travel with a clean and interactive user interface.

## 🚀 Features

* 🔍 Search Images
* 🏷️ Category Filters
* 🌙 Dark / Light Mode
* ❤️ Favorite Images
* ⬇️ Download Images
* 📤 Image Upload Feature
* 🖼️ Full-Screen Image Preview
* 📱 Fully Responsive Design
* 💾 Local Storage Support
* ♾️ Infinite Scroll Foundation
* 🎨 Modern Glassmorphism UI
* 📖 About Section

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* JSON
* Local Storage API
